  ### Dit ging goed/Heb ik geleerd: 
  Het leren omgaan met HTML en de verschillende selectoren toepassen ging erg goed. In het begin vond ik het lastig om verschillende elementen te selecteren zonder classes of ID's. Maar nu kan ik meer selecteren in het algemeen zonder dat te gebruiken. Daarnaast ben ik meer bezig geweest met het uittesten van verschillende methodes van plaatsing, in mijn geval vooral position en soms grid. Animatie is beter gegaan met hulp van de docent. Als het gaat om positionering in het algemeen ging dat wel veel beter dan de eerste keren dat ik met code heb gewerkt. Het plaatsen van objecten ging een stuk makkelijker dan normaal. Mijn code is ook meer overzichtelijker dan normaal. Ik denk dat een andere goede dingen die ik geleerd heb is het werken met slideshows en hoe je fotos op verschillende manieren kan visualiseren (Waar ik ook moeite mee had)


  ### Dit was lastig/Is niet gelukt:

<bold> Wat vond ik lastig </bold>
Wat ik lastig vond was het symantisch maken van Javascript. Javascript is niet mijn sterkste kan van coderen en is ook iets waar ik echt goed aan moet werken. Mijn moeites met Java is het symantische code laten maken en wat het beste manier is van Java te gebruiken. Tijdens mijn lessen heb ik telkens meer geleerd dat CSS veel meer dingen kan doen dan ik verwacht had. Wat ik daarin lastig vind is het bepalen wanneer ik java moet gebruiken en wanneer ik css kan gebruiken

Wat ik ook lastig vond was het onderzoeken naar een oplossing. Een voorbeeld hiervan is mijn carasoul, online staat veel info hoe je een carasoul moet maken, maar veel van de voorbeelden zijn niet symantisch. Ik vind het dan lastig om te kijken hoe ik precies het symantisch kan maken of ik just verder moet zoekn naar een alternatief. Voor mij is het handig voor de volgende keer een eigen cheat sheet te maken op basis van de kennis die ik geleerd heb, maar ook begrijp

Daarnaast was het maken van complexe onderdelen zoals hamburger menu's en carasouls erg lastig. Niet omdat ik er geen uitleg voor heb gekregen, want de lessen gaven alle info die ik nodig had. Maar omdat ik het lastig vond toetepassen in mijn werk.
